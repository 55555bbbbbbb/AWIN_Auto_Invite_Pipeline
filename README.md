# AWIN KOL Automation Pipeline (Awin KOL 自動開發工具)

這是一個針對 Awin 聯盟行銷平台開發的自動化通訊管線。使用者只需輸入指定關鍵字，程式即可自動搜尋相關的 KOL（意見領袖），過濾重複名單，並批次自動發送合作邀請訊息，大幅降低網紅開發的人力時間成本。

## 系統架構

* **main.py**: 程式執行主入口，負責調度各個模組。
* **config.py**: 參數設定檔（包含搜尋關鍵字、Awin 平台登入資訊與延遲設定）。
* **grab.py**: 資料擷取模組，負責在 Awin 平台上爬取符合關鍵字的 KOL 名單。
* **dedupe.py**: 資料去重模組，清洗已爬取的名單，避免對同一位 KOL 重複發送訊息。
* **send.py**: 訊息發送模組，執行自動化點擊與發送動作。
* **utils.py**: 通用輔助函式庫（如瀏覽器驅動設定、錯誤處理）。
* **invite_message.txt**: 合作邀請文本模板，可自由抽換發送內容。
